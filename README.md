# Tanks-online

For now, the project is still in WIP state, level editor and basic game mechanics are mostly finished. In the future, bots and multiplayer system will be added. 

Controls:
* W - move forward
* S - move backward
* D - rotate clockwise
* A - rotate counterclockwise
* Space - Shoot
